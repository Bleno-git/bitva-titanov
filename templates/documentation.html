ENDPOINT Описание Метод Параметры 
/api/version/
 Используется для автоматического получения версии API. Каждая новая версия обратно совместима. 
GET
 
Параметров нет
 
/api/upload/
 Загрузка файлов с данными для аналитики 
POST
 
type - тип загружаемых данных
data - файл с данными
file_name - тип данных. Доступные типы - ['cost_data', 'shipping_data_fact', 'shipping_data_plan']
